# Yuge Wang — Game Designer Portfolio

A personal portfolio website for showcasing game design projects, skills, and experience.

## Tech Stack

- HTML5
- CSS3 (custom properties, Grid, Flexbox, animations)
- Google Fonts (Inter)
- No frameworks — lightweight and fast

## Structure

```
index.html          Main single-page site
css/style.css       All styles
assets/images/      Project screenshots & avatar
```

## Getting Started

Open `index.html` in any modern browser. No build step required.

## Customization

1. Replace placeholder project content in `index.html` with your real projects
2. Add screenshots to `assets/images/` and update `<img>` tags
3. Update contact links (email, LinkedIn, GitHub) with your real URLs
4. Swap the avatar placeholder with your photo
